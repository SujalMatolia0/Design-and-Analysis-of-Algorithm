# Big Theta Notation(Θ)

It is also called as optimal bound or tightest bound

### Defination

$f(n) = equal to Θ(g(n)) if and olny if

$f(n) =Θ(g(n))$ and $f(n) = Ω(g(n))$

### Example

$f(n) = n^2 + n + 1$

then $f(n) = Θ(g(n))$

previously we proved that

$f(n) = Θ(n^2)$ and $f(n) = Ω(n^2)$

so that $f(n) = Θ(n^2)$

<Alert title="Note">
  If $f(n)= C_0+C_1n+C_2n^2+C_3n^3+...+C_mn^m$
  
  where $(C_m ≠ 0)$

    then $f(n) = O(n^m)$

         $f(n) = Ω(n^m)$

         $f(n) = Θ(n^m)$

<br/>
  If the function is polynomial of the degree 'm' then the function is $O(n^m)$, $Ω(n^m)$, $Θ(n^m)$ 
<br/>
 If your function is greater than your polynomial value then use dominals ranking concept.
</Alert>
